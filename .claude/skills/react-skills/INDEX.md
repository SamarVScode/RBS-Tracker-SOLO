# react-skills — Skill Index

5 skills for React + Vite projects. Each has SKILL.md (always-load) + REF.md (lazy-load).
Load only what the current task needs.

## Skill Map

| Skill | Load When | Path |
|---|---|---|
| `react-routing` | Creating pages, routes, ProtectedRoute, react-router-dom v6 | `routing/` |
| `react-data-fetching` | TanStack Query setup, SWR, service layer patterns | `data-fetching/` |
| `react-assets` | Vite asset handling, env vars, static files, svg imports | `assets/` |
| `react-testing` | Vitest + RTL full setup, test utilities | `testing/` |
| `react-upgrade` | React 18→19, Vite upgrades, breaking changes | `upgrade/` |

## Load Combinations by Task

| Task | Skills to Load |
|---|---|
| Build a new page | `react-routing` |
| Add data fetching | `react-data-fetching` |
| Add images/assets | `react-assets` |
| Write tests | `react-testing` |
| Upgrade React/Vite | `react-upgrade` |
| Full feature | `react-routing` + `react-data-fetching` |
