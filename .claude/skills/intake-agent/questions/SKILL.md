# Questions by Task Type

## UI Feature
- What fields/inputs does it need?
- One record per user per day, or multiple?
- Where does it appear? (new page / dashboard widget / tab / modal)
- Is past data editable?
- Any validation rules?

## Bug Fix
- What is the exact error message or unexpected behaviour?
- Which screen/flow triggers it?
- Is it reproducible every time or intermittent?
- What did you expect to happen?

## Refactor
- What problem does the current code cause?
- Which files are in scope?
- Any behaviour that must NOT change?

## Performance
- What specific metric is slow? (load time, bundle size, FPS)
- Where does it happen? (specific screen/route)
- Do you have measurements? (Lighthouse score, actual ms)

## New Project (first session)
- What is the app name?
- React (Vite) | Next.js | React Native / Expo?
- Firebase project ID?
- Primary color and background color?
- Styling approach: inline styles | Tailwind | CSS modules | StyleSheet.create?

## Rules
- Batch ALL questions in ONE message — never ask one at a time
- Maximum 5 questions total
- Never ask what README already states
- Never ask platform if mentioned anywhere in prompt or README
